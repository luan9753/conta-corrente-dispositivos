# Conta Corrente de Dispositivos

Pagina independente do pacote Aura original para acompanhamento da conta corrente de dispositivos.

Arquivos principais:

- `CONTA_CORRENTE_DISPOSITIVOS.html`
- `CONTA_CORRENTE_DISPOSITIVOS_DATA.js`
- `CONTA_CORRENTE_DISPOSITIVOS_CONFIG.json`
- `gerar_conta_corrente_dispositivos.py`
- `validar_conta_corrente_dispositivos.py`
- `RELATORIO_VALIDACAO_CONTA_CORRENTE.md`
- `ATUALIZAR_CONTA_CORRENTE_10_MIN.bat`

Atualizacao local:

```bat
ATUALIZAR_CONTA_CORRENTE_10_MIN.bat
```

A rotina executa geracao, validacao, commit e push apenas deste projeto. O pacote Aura original permanece como fonte de leitura.
